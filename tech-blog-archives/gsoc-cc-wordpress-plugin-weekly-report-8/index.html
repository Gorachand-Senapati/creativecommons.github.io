GSoC CC WordPress Plugin: Weekly Report #8

by erlehmann on Wednesday, July 21, 2010

This is a short one. This week, I did only a few things related to the plugin: First, planned features and bugs are now listed in the issue tracker. Second, you can now specify default attribution in the form of rights holder and URL. I did also work on the default licensing mechanism, but it probably needs a complete overhaul.

For this week, I will be looking into more REST API stuff, get rid of the shortcodes so even with a deactivated plugin, there will still be content (parsing the elements Wordpress inserts instead, using DOM methods) and approach the mentioned rewrite of the default licensing mechanism.
